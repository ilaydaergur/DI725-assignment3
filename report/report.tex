\begin{abstract}
Large Vision-Language Models (VLMs) show strong general-purpose performance but require
domain-specific fine-tuning to perform well on specialised tasks such as remote sensing image captioning.
In this work, we apply QLoRA~\cite{dettmers2023qlora} --- 4-bit quantisation combined with
low-rank adapters --- to fine-tune PaliGemma-3B~\cite{beyer2024paligemma} on the RISC dataset,
training only 0.39\% of the model's parameters.
Our approach achieves substantial gains over the pre-trained baseline:
BLEU-4 improves from 0.0062 to 0.0553 ($\times$9), METEOR from 0.0897 to 0.2285,
and ROUGE-L from 0.1476 to 0.2675.
\end{abstract}

\section{Introduction}

Remote sensing image captioning is a challenging vision-language task requiring models to
describe satellite imagery using precise, domain-specific vocabulary.
PaliGemma~\cite{beyer2024paligemma} is a 3B-parameter VLM combining the SigLIP
vision encoder with the Gemma language decoder, pre-trained on diverse image-text pairs.
Although capable across many benchmarks, it lacks familiarity with satellite imagery terminology
and produces generic outputs without adaptation.

QLoRA~\cite{dettmers2023qlora} quantisizes the frozen base model weights
to 4-bit precision while training a small set of low-rank adapter (LoRA) matrices.
This reduces trainable parameters to less than 1\% of the total, enabling
high-quality fine-tuning on a single GPU.

\section{Method}

\subsection{Dataset}

We use the RISC dataset~\cite{riscdata} (\texttt{caglarmert/full\_riscm}),
comprising 44,521 satellite images at 224$\times$224\,px resolution,
each annotated with five human-written captions covering diverse remote sensing scenes.
The dataset is split 95/5 into 42,294 training image--caption pairs and 2,226 test images.
Evaluation is performed on a 200-sample subset using one ground-truth caption per image,
as required by the assignment specification.

\subsection{Model and QLoRA Setup}

PaliGemma-3B (\texttt{google/paligemma-3b-pt-224}) is loaded in 4-bit NF4
quantisation using BitsAndBytes with \texttt{bfloat16} compute dtype.
LoRA adapters with rank $r\!=\!8$ are attached to all seven
linear projection layers of the Gemma language decoder:
\texttt{q\_proj}, \texttt{k\_proj}, \texttt{v\_proj}, \texttt{o\_proj},
\texttt{gate\_proj}, \texttt{up\_proj}, and \texttt{down\_proj}.
The SigLIP vision encoder and the multi-modal projector are frozen throughout,
focusing adaptation capacity on language generation.
This yields \textbf{11.3M trainable parameters out of 2.93B total (0.39\%)},
as reported by \texttt{model.print\_trainable\_parameters()}.

Training uses the AdamW-8bit optimiser ($\beta_2\!=\!0.999$),
learning rate $2\!\times\!10^{-5}$,
effective batch size 16 (4 per device $\times$ 4 gradient accumulation steps),
100 warm-up steps, and \texttt{bfloat16} mixed precision for 1 full epoch.
Each input is formatted as the text prompt \textit{``describe this image.''}
paired with the satellite image; the corresponding caption serves as the generation target.
The processor produces input tokens with image embeddings prepended,
and cross-entropy loss is computed only on caption tokens.

\section{Results and Discussion}

\subsection{Quantitative Evaluation}

Table~\ref{tab:results} and Fig.~\ref{fig:metrics} report BLEU-4, METEOR, and ROUGE-L
on 200 held-out test images.
All three metrics improve consistently after QLoRA fine-tuning.
BLEU-4 increases nearly 9$\times$, from 0.0062 to 0.0553.
METEOR and ROUGE-L, which better account for synonym overlap and recall,
both roughly double (0.0897$\to$0.2285 and 0.1476$\to$0.2675, respectively).
Low absolute BLEU-4 values are typical for single-reference captioning evaluation
and are consistent with reported scores on standard benchmarks under the same protocol.

\section{Conclusion}

We demonstrated that QLoRA enables effective domain adaptation of PaliGemma-3B with only
0.39\% trainable parameters, using a single A100 GPU for one training epoch.
Fine-tuning on the full RISC dataset yields large quantitative improvements across
BLEU-4, METEOR, and ROUGE-L, and produces qualitatively richer, domain-specific captions
for remote sensing imagery.
These results confirm that parameter-efficient fine-tuning is a viable and practical
strategy for adapting large VLMs to specialised vision-language tasks without
incurring the cost of full fine-tuning.
